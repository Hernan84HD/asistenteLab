
# Preparación de Soluciones para Análisis de Laboratorio

## Análisis de Proteínas

**H2SO4 0.1 N (4 Litros)**  
- 11.2 ml de H2SO4 98 % en 4000 ml de H2O

**H2SO4 0.1 N (2 Litros)**  
- 5.6 ml de H2SO4 98 % en 2000 ml de H2O

**Titulación:**  
- **Patrón:** Na2CO3  
- **Indicador:** Heliantina  

1. En un Erlenmeyer, pesar aproximadamente 0.1 gr del patrón.
2. Disolver con 25 ml de agua destilada.
3. Agregar 3 gotas de indicador.
4. Titular con la solución H2SO4 0.1 N hasta que vire de rojo a naranja/amarillo.
5. Realizar por triplicado, más un blanco.

**Cálculo de normalidad:**  
N = (1000 x gr de Na2CO3 x pureza de Na2CO3 / 100) / (53 x (ml H2SO4 – ml Blanco))

## Análisis de Acidez

**La solución de NaOH 0.1 N ya viene preparada en botellas de 1 litro.**

**Titulación:**  
- **Patrón:** Biftalato de Potasio  
- **Indicador:** Fenolftaleína  

1. En un Erlenmeyer, pesar aproximadamente 0.1 gr del patrón.
2. Disolver con 25 ml de agua destilada.
3. Agregar 3 gotas de indicador.
4. Titular con la solución NaOH 0.1 N hasta que vire de incoloro a rosa pálido.
5. Realizar por triplicado, más un blanco.

**Cálculo de normalidad:**  
N = (1000 x gr de patrón x pureza del patrón / 100) / (204.23 x (ml NaOH – ml Blanco))
